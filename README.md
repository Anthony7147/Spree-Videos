SpreeVideos
===========

Display video(s) on your product page.

* Pulls thumbnails from youtube
* Clickable thumbnails to easily switch between videos
* Versions available for Spree 0.7 & 1.0

Example
=======
In `products/show`:

	<%= render 'video', :product => @product %>
